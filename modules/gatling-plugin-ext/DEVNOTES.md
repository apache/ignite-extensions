# Ignite Gatling Plugin

## Build

```shell
mvn clean package -DskipTests
```

## Run tests

```shell
mvn test
```

## Coding style

Check coding style

```shell
mvn scalastyle:check -Pcheckstyle
```


Preview automatic code formatting changes:

```shell
mvn spotless:check
```


Apply automatic code formatting:

```shell
mvn spotless:apply
```
